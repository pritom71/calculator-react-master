Simple Project Calculator with Vite + ReactJs

## Installation
To run this project, follow these steps:
   ```bash
   git clone https://github.com/ilsyaa/calculator-react.git

   cd calculator-react

   npm install

   npm run dev
   ```

## build product
  ```bash
  npm run build
  ```
